## 참고

- 개발 서버 실행: `npm run dev` (포트는 자동 — 보통 5173~5175)
- 자원 지갑은 `useResourceStore`(src/state)가 관리하며, `useDataStore`의
  비동기 로드가 끝난 뒤 `resources.json`의 `start` 값으로 1회 하이드레이션됨
  — MOD-8에서 자원 흐름을 건드릴 때 이 스토어를 그대로 확장하면 됨.
- 모듈이 끝나면 직접 `npm run dev`로 실행해 확인 → 문제가 있으면
  **에러 메시지를 복붙**하고 "이 에러 고쳐줘"라고 요청하세요.
